CONTRIBUTING

This is a private, proprietary repository. Contributions are only accepted by prior
written permission from the repository owner, Divesh Kumar S.

If you wish to contribute:

1. Request permission using the instructions in `README.md`.
2. If permission is granted, you may be required to sign a Contributor License Agreement (CLA) or NDA.
3. Follow the code style and testing requirements in this repo. Run existing tests before submitting.

When submitting a change:
- Create a fork or branch as instructed in your approval email.
- Include a short description of the change, rationale, and any tests added.
- Do not include any secrets, passwords, or private data.

If you are not explicitly authorized, do not open pull requests or attempt to run the code.

Contact: divesh@example.com
